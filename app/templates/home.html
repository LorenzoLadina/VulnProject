In this section a bank's web application is simulated, the aim of the challenge is to be able to obtain the three flags, an example of a flag is the following: {TH!S-IS-A-Fl4g}.
Each flag should be obtained by exploiting the following web attacks: CSRF, Cookie-Bruteforce and Data-Exfiltration.
